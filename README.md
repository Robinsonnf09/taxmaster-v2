# 🚀 Tax Master v2.0

Sistema profissional de gestão de precatórios com automação inteligente.

## 📋 Recursos

- ✅ **Calculadora de Precatórios** - Cálculos precisos com juros e correção monetária
- ✅ **Dashboard Futurista** - Interface moderna e responsiva
- ✅ **Gestão de Processos** - Controle completo de precatórios
- ✅ **Automação Inteligente** - Busca e atualização automática
- ✅ **Relatórios** - Geração de relatórios profissionais

## 🧮 Calculadora Profissional

- Juros simples e compostos
- Correção monetária (IPCA, INPC, TR, SELIC)
- Detalhamento mês a mês
- Cálculo proporcional de dias
- Anos bissextos

## 🛠️ Tecnologias

- **Backend:** Python + Flask
- **Frontend:** HTML5, CSS3, JavaScript
- **Banco de Dados:** SQLite
- **Deploy:** Railway.app
